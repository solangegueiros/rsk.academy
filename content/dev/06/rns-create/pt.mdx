## O que é RNS - RIF Name Service?

RNS é um projeto open source ue proporciona uma arquitetura para identificação de endereços blockchain através de nomes de fácil leitura.

O conceito de um serviço para resolução de nomes facilita o uso de endereços para a utilização por usuários desbancarizados e que não são técnicos. 

- Na Mainnet, adquirir um nome de domínio RNS custa menos de um dólar.
- Uma vez que o domínio for adquirido, é possível criar e delegar subdomínios sem custo.
