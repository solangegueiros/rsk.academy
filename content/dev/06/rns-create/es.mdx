## ¿Qué es RNS - RIF Name Service (Servicio de nombres RIF)?

RNS es un proyecto de código abierto que proporciona una arquitectura para identificar direcciones de blockchain a través de nombres fáciles de leer.

El concepto de un servicio de resolución de nombres facilita el uso de direcciones para usuarios no bancarizados y no técnicos.

- En Mainnet, adquirir un nombre de dominio RNS cuesta menos de un dólar.
- Una vez adquirido el dominio, es posible crear y delegar subdominios sin costo alguno.
