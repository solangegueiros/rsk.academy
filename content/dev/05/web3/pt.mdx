Nesta lição vou apresentar o que é **Web3.js** e como usá-lo para se comunicar com o Blockchain RSK.

Web3.js é uma coleção de bibliotecas.
É a principal biblioteca JavaScript para interagir com o Ethereum Blockchain
e também funciona para o RSK Blockchain.
Permite que você interaja com um nó RSK / ethereum local ou remoto usando HTTP, IPC ou WebSocket.

Estamos falando sobre o desenvolvimento de sites ou clientes que interagem com o Blockchain,
ou seja, escrever código que lê e grava dados de contratos inteligentes no Blockchain.

Os programas externos podem interagir com o cliente EVM usando a interface web3.
As bibliotecas Web3 expõem vários métodos baseados em JSON RPC que podem ser usados para interagir com os clientes Ethereum e RSK.

Web3.js permite:

- enviar Ether de uma conta para outra
- ler e gravar dados de contratos inteligentes
- criar contratos inteligentes
- e muito mais!

Web3.js se comunica com o Blockchain usando protocolo JSON RPC (Remote Procedure Call).
Ele nos permite fazer solicitações a um nó RSK / Ethereum individual com JSON RPC para ler e gravar dados na rede.

Web3.js tem uma comunidade de usuários e mantenedores associados à Fundação Ethereum.
Tem uma boa referência para sua API.
Ele existe desde 2015 e foi amplamente utilizado por muitos projetos.

## RSK3.js

A biblioteca rsk3.js é a versão RSK do web3.js.
Permite que você interaja com o próprio blockchain RSK e os com contratos inteligentes publicados nesta rede.
