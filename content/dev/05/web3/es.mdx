En esta lección, te mostraré qué es **Web3.js** y cómo usarlo para comunicarse con Blockchain RSK.

Web3.js es una colección de bibliotecas.
Es la principal biblioteca de JavaScript para interactuar con Ethereum Blockchain.
y también funciona en RSK Blockchain.
Le permite interactuar con un nodo RSK / ethereum local o remoto utilizando HTTP, IPC o WebSocket.

Estamos hablando de desarrollar sitios web o clientes que interactúen con el blockchain,
significa escribir código que lee y escribe datos de la cadena de bloques con contratos inteligentes.

Los programas externos pueden interactuar con el cliente de EVM utilizando la interfaz web3.
Las bibliotecas Web3 exponen varios métodos basados en JSON RPC que se pueden utilizar para interactuar con los clientes Ethereum y RSK.

Web3.js te permite:
- enviar Ether de una cuenta a otra
- leer y escribir datos de contratos inteligentes
- crear contratos inteligentes
- ¡y mucho más!

Web3.js se comunica con Blockchain mediante el protocolo JSON RPC (Remote Procedure Call).
Nos permite realizar solicitudes a un nodo RSK / Ethereum con JSON RPC para leer y escribir datos en la red.

Web3.js tiene una comunidad de usuarios y mantenedores asociados con la Fundación Ethereum.
Tiene una buena referencia para su API.
Ha existido desde 2015 y ha tenido un uso extensivo en muchos proyectos.

## RSK3.js

La biblioteca rsk3.js es la versión RSK de web3.js.
Le permite interactuar con la Blockchain RSK en sí y con los contratos inteligentes implementados en esta red.
